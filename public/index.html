My Work

Palate Recipe App

This app allows a user to search for and discover recipes. The 'Saved Recipes' route is protected until the user signs up and becomes authenticated. Once the user is authenticated and logged in, they can save and delete their unique recipes. The recipe data is being pulled from the Edamam API and user data is being stored in Firebase. I also used React, React Router, Semantic UI, and CSS to build the user interface and to populate the page with recipes.

Giphy Search Engine

This web app allows you to search for a specific GIF and pick how many GIFs you want to be displayed on the page. There is also a route for popular GIFs and for a random GIF. This was a fun, personal project I built to become more familiar with getting data from an external API. I built this app with React, React Router, Material-UI, CSS, HTML, and the Giphy API. It is hosted on Netlify.
